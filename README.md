# 🌐 ShareSphere – File Sharing System

## 📌 Overview
ShareSphere is a scalable file sharing system built using Streamlit, Python, and MongoDB (NoSQL). It allows users to upload, store, and access files securely.

## 🚀 Features
- 📂 File Upload & Download
- 🔐 Secure File Handling
- 📊 Monitoring Dashboard
- ⚡ Optimized MongoDB Queries
- 🌍 Scalable Architecture

## 🛠 Tech Stack
- Frontend: Streamlit
- Backend: Python
- Database: MongoDB (NoSQL)

## 📁 Project Structure
file-sharing-system/
│── app.py                # Streamlit UI + main logic
│── backend/              # Backend logic
│   ├── db.py             # MongoDB connection
│   ├── file_handler.py   # File upload/download logic
│── uploads/              # Stored files
